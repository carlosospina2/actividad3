# 🌌 AstroCosmos - Destinos de Astroturismo

AstroCosmos es una landing page moderna y responsiva dedicada a la exploración del cielo nocturno y la promoción de destinos turísticos astronómicos. Este proyecto es una adaptación visual inspirada en un diseño de ecoturismo, optimizada para ofrecer una experiencia de usuario inmersiva con una temática espacial.

---

## 🚀 Características

* **Diseño Responsivo:** Adaptado para dispositivos móviles, tablets y escritorio.
* **Menú Hamburguesa:** Implementado puramente con CSS/SCSS (sin dependencias pesadas de JS).
* **Estructura BEM:** Código limpio y escalable siguiendo la metodología *Block Element Modifier*.
* **Arquitectura SCSS:** Uso de variables, anidamiento y mixins para un mantenimiento sencillo.
* **Galería Dinámica:** Layout basado en CSS Grid para una visualización moderna de imágenes espaciales.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **SCSS (Sass):** Estilos avanzados y modulares.
* **Google Fonts:** Tipografías para una lectura clara.
* **Unsplash API:** Imágenes de alta resolución para la temática espacial.
